# 🔢 Basic Calculator - CodeAlpha Task 2

A responsive and interactive web-based calculator built using HTML, CSS, and JavaScript.

## 🌟 Features
* **Arithmetic Operations:** Supports Addition (+), Subtraction (−), Multiplication (×), and Division (÷).
* **Real-time Display:** Immediate feedback for user inputs and calculations.
* **Clear & Delete:** Functions to clear the entire screen (AC) or delete the last digit (DEL).
* **Keyboard Support:** (Bonus) Users can use their keyboard to input numbers and operators.
* **Styling Enhancements:** Modern UI with smooth button transitions and hover effects.

## 🛠️ Built With
* **HTML5:** For the structure.
* **CSS3:** For custom styling and layout.
* **JavaScript (ES6):** For calculation logic and keyboard event handling.
